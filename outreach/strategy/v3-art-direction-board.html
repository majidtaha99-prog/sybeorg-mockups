SYBEORG / v3 Mockup Direction
Purpose: get art direction right before updating production pages
No mass recolors. No contractor-template slop.

Five separate design worlds before we touch the five pages.

This board turns the M&P v3 lesson into a stricter rollout system: each prospect gets its own mood, visual motif, interaction, and money-leak fix. If a page cannot explain why it belongs to that exact business, it does not ship.

Ship gate

  • Custom stance: business-specific visual language, not a reused hero/form/card stack.
  • Real proof only: no fake numbers, badges, testimonials, licenses, or owners.
  • Mobile action: phone/service path visible within the first screen.
  • One interaction: selector, route, dispatch board, proof ledger, or service triage.
  • Visual QA: mobile + desktop screenshot pass before commit/push.
